BookBot

BookBot was created by me when I was quite new to python, I made it to practice my skills in python
You pass the path of a .txt file saved in you device locally and it gives you the option to perform 2 operations on it, the first one gives the total amount of characters in the text as well as the count of each character in a descending order. The second option summarizes the content of the file.

As mentioned the code is written entirely in python and I have used Gemini API to summarize the text
